Du bist am Rechner. Die Anleitung funktioniert genauso – nur die Homescreen-Verknüpfung am Ende entfällt.

Alle Knöpfe öffnen einen neuen Tab – diese Anleitung bleibt offen. Tippe auf eine Überschrift, um einen Schritt abzuhaken.

  1. Firefox installieren

    Chrome kann keine Erweiterungen. Dein Standardbrowser bleibt unverändert.

    Firefox laden

  2. Tampermonkey hinzufügen

    In Firefox: Menü ⋮ → Add-ons → Tampermonkey → Hinzufügen.

    Tampermonkey öffnen

  3. Skript installieren

    Tampermonkey zeigt ein Installationsfenster – dort auf „Installieren" tippen.

    Skript installieren

  4. Auf den Startbildschirm legen

    Öffnet die Oberfläche direkt beim Antippen.

    Oberfläche öffnen

    Dort dann: Menü ⋮ → Zum Startbildschirm hinzufügen.
